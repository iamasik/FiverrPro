# Fiverr Plus

**Fiverr Plus by EMLinter** - Powerful toolkit for Fiverr freelancers to increase productivity, discover analytics, ranking insights, and AI services for gigs.

## Features

- **Notification Alerts**: Get real-time notifications for new messages and important updates
- **Auto Refresh**: Automatically refresh pages with customizable intervals or random refresh mode
- **Hide Balance**: Privacy feature to hide your account balance from prying eyes
- **Hide Order Price**: Keep order prices private for enhanced security
- **Dark Mode**: Eye-friendly dark theme for comfortable browsing
- **Keyword Analytics**: Discover keyword analytics and insights for your gigs
- **Gig Ranking**: Track and analyze your gig ranking performance

## Installation

1. Clone this repository
2. Load the extension in your browser (Chrome/Edge)
3. Enable the extension and start using Fiverr Plus features
