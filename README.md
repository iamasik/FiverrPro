# Fiverr Pro

Fiverr Pro - Toolkit for Fiverr Freelancers to increase productivity, discover analytics, ranking and AI services for gigs.

## Features

- **Notifications & Message Alerts**: Stay updated with real-time notifications
- **UI Enhancements & Shortcuts**: Enhanced user interface with convenient shortcuts
- **Hide My Balance**: Privacy feature to hide your balance
- **Dark Mode**: Eye-friendly dark theme

## Installation

1. Clone this repository
2. Load the extension in your browser (Chrome/Edge)
3. Enable the extension and start using Fiverr Pro features
